# Datable — 腐らないデータ基盤を学ぶ無料コース

> 誰でも入口に立てて、修了する頃には「**作ったけど使われない／価値がチームに閉じる／想定外の使い方をされる／使われすぎて変更できない**」を防ぐデータ基盤を、ゼロから設計・実装できるようになる無料の教育サイトです。

🌐 **公開URL**: https://kenta-suzuki-coconala.github.io/data-platform-academy/

## 特徴

- **「腐らない」データ基盤に焦点** — よくある4つの失敗モードを背骨に、基礎からモデリング・パイプライン・品質・設計・実践まで体系的に学べる **24レッスン / 7パート**。
- **ブラウザだけで完結** — 登録不要・完全無料。ビルド不要の静的サイト。
- **ブラウザ内SQL Playground** — [sql.js](https://github.com/sql-js/sql.js)（SQLite/WASM）でサンプルECデータに対し本物のSQLをその場で実行。サーバー送信は一切なし。
- **学習体験にこだわったUI/UX** — ダーク/ライトテーマ、学習進捗（ローカル保存）、レッスン内コードを Playground にワンクリック転送、Mermaid 図、シンタックスハイライト。

## ローカルで動かす

ビルド不要。任意の静的サーバーで配信するだけ。

```bash
python3 -m http.server 8000
# → http://localhost:8000/
```

> `file://` ではなく **HTTP** で開いてください（`fetch` でレッスン Markdown を読み込むため）。

## 構成

```
.
├── index.html            # SPA シェル（ハッシュルーティング）
├── css/style.css         # デザインシステム
├── js/
│   ├── app.js            # ルーター / Markdown描画 / 進捗 / テーマ
│   └── playground.js     # sql.js による SQL Playground
├── content/
│   ├── manifest.json     # カリキュラム定義（ナビ生成元）
│   └── lessons/*.md      # 各レッスン（Markdown）
└── data/
    ├── gen_seed.mjs      # サンプルデータ生成スクリプト（決定的）
    └── seed.sql          # Playground 用サンプルデータ
```

### コンテンツの追加・編集

1. `content/lessons/<id>.md` を追加または編集する。
2. `content/manifest.json` の該当パートに `{ "id", "title", "minutes" }` を追加する。

Markdown はコールアウト（`:::insight … :::`）と Mermaid 図に対応しています。

### サンプルデータの再生成

```bash
node data/gen_seed.mjs   # → data/seed.sql
```
